# CRUD CHALLENGE

Building a REST API using express.

## About the Challenge

You should make a backend implementation using express.

## Getting started

        # clone repo
        $ git clone https://github.com/matheustanaka/crud-challenge.git

        # install dependencies
        $ yarn

        # start project
        $ npm run start:dev

        # you can see the project on localhost
        http://localhost:8000/users

## About the API

Create a Rest API with 2 models, first User Model and second Medicines Model. 
The API is basically a simple wallet where you can manage your medicines.

### Api

-   [x] GET User
